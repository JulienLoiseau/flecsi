Detailed Description

template<size_t... PRIVILEGES>
struct flecsi::privilege_pack< PRIVILEGES >

Utility type to allow general privilege components that will match the old style of specifying permissions, e.g., <EX, SH, GH> (The old approach was only valid for mesh type topologies, and didn't make sense for all topology types). A terminator with value 0x02 is appended to the privileges provided in the template pararmeter so that a count can be determined.
