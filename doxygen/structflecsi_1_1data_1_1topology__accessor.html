Detailed Description

template<typename TOPOLOGY_TYPE, size_t PRIVILEGES>
struct flecsi::data::topology_accessor< TOPOLOGY_TYPE, PRIVILEGES >

Topology accessor type. Topology accessors are defined by the interface of the underlying, user-defined type, i.e., unlike storage class accessors, topologies can be customized by the specialization to add types and inerfaces that are not part of the core FleCSI topology type. By inheriting from the customized topology type, we pick up these additions.
