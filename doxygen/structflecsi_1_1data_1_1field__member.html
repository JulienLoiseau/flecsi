Detailed Description

template<typename DATA_TYPE, storage_label_t STORAGE_CLASS, typename TOPOLOGY_TYPE, size_t INDEX_SPACE>
struct flecsi::data::field_member< DATA_TYPE, STORAGE_CLASS, TOPOLOGY_TYPE, INDEX_SPACE >

The field_member type provides a mechanism to define and register fundamental field types with the FleCSI runtime.

Template Parameters
DATA_TYPEA compact type that will be defined at each index of the associated topological index space. FleCSI defines a compact type as either a P.O.D. type, or a user-defined type that does not have external references, i.e., sizeof for a compact type is equal to the logical serialization of that type.
STORAGE_CLASSThe label of the storage class for the field member.
TOPOLOGY_TYPEA specialization of a core FleCSI topology type.
INDEX_SPACEThe id of the index space on which to define the field.
